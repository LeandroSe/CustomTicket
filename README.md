CustomTicket
------------

Sistema de ticket customizavel.

Instalação
----------

```
git clone git@github.com:LeandroSe/CustomTicket.git
composer install
```

Execução do PHP Built-in Server - Development
---------------------------------------------

```
php -S localhost:8000 web/ web/index_dev.php
```
